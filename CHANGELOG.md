# Changelog

All notable changes to Paper2Repo are documented here.

## v0.1.0 - 2026-05-16

Initial public release.

- Local-first PDF and arXiv paper analysis workspace.
- FastAPI backend, Next.js frontend, LangGraph analysis workflow, and SQLite persistence.
- Evidence-grounded reports covering paper understanding, method audit, experiment audit, reproduction risks, and code skeleton planning.
- Recoverable analysis pipeline with deterministic parsing and report-generation fallbacks.
- OpenAI-compatible chat model support for richer analysis and Q&A.
- Batch analysis, citations, knowledge search, comparison, HTML/PDF/LaTeX export, and report downloads.
- Docker Compose setup for a faster local trial.
